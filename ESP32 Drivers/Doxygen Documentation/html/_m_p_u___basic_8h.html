◆ mpuInit()

esp_err_t mpuInit ( uint8_t  AD0)

This function intializes the IMU by setting the value of the AD0 bit in its address, Initializing the I2C port chosen, and then waking up the IMU from sleep mode.

Parameters
AD0This variable reflects the state of the AD0 pin on the MPU6050 where 1 means its connected to VCC or 0 when its connected to GND.
Returns
esp_err_t Will return ESP_OK upon successful completion or an error otherwise.

◆ wakeUp()

esp_err_t wakeUp ( )
